# 📝 TODO-LIST

A simple and responsive **To-Do List Application** built with **HTML, CSS, and JavaScript**, using **LocalStorage** to persist tasks.
## ✨ Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Data persistence using **localStorage**
- Responsive design for mobile and desktop
- Alerts with smooth animations for actions

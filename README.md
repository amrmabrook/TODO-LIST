# 📝 TODO-LIST

A simple and responsive **To-Do List Application** built with **HTML, CSS, and JavaScript**, using **LocalStorage** to persist tasks.

## 🚀 Features
- Add new tasks with a title.
- Edit existing tasks.
- Delete tasks.
- Mark tasks as completed or uncompleted.
- Store tasks in LocalStorage so they remain after page reload.
- Alerts with smooth animations for actions (add, delete, error).
- Responsive design for both desktop and mobile.

## 🛠️ Technologies Used
- **HTML5** – Structure of the app.
- **CSS3** – Styling, animations, and responsive layout.
- **JavaScript (Vanilla)** – Core logic for adding, editing, deleting, and updating tasks.
- **LocalStorage** – Persistent storage in the browser.
